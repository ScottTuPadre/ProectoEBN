Glosario

Adaptativo: Perteneciente o relativo a la adaptación o a la capacidad de adaptación.

BleachBit: Es un limpiador de código libre que permite eliminar archivos temporales liberando espacio en el disco duro.

Disco Duro: Un disco duro en un dispositivo de almacenamiento de datos no volátil que emplea un sistema de grabación magnética para almacenar datos digitales de forma rápida y seguridad.

Giga byte: Es una unidad estándar usada con frecuencia en informática. aporta información sobre la capacidad de almacenamiento de un medio o el tamaño de un archivo.

Gigahercio: El gigahercio es un múltiplo de la unidad de medida de frecuencias hercio y equivale a 109 Hz.

Hardware: Es el conjunto de componentes físicos de los que están hecho el equipo.

Linux: Es un sistema gratuito, personalizable y mas seguro.

Memoria RAM: Es la memoria a corto plazo de un ordenador donde se almacenan los datos que el procesador esta utilizando es ese momento.

Preventivo: que previene un mal o un peligro o sirve para prevenir.

Procesador: es el cerebro del sistema, justamente procesa todo lo que ocurre en la PC y ejecuta todas las acciones que existen.

Requisitos técnicos: son especificaciones en detalles de los contenidos y exigencias establecidas para cada etapa de desarrollo.

ROOT: es el usuario que tiene acceso administrativo al sistema.

Rufus: es una utilidad que el ayuda a formatear y crear soportes USB de arranque como pendrive, tarjetas de memoria, etc.

Software: es el conjunto de programas o aplicaciones, instrucciones y reglas informáticas que hacen posible el funcionamiento del equipo.

USB: una memoria USB (Universal Serial Bus) es un dispositivo de almacenamiento masivo que utiliza memoria flash para guardar la información que puede requerir.

Windows 7: es una versión de Microsoft Windows, línea de sistemas operativos producida por Microsoft corporation. Se lanzo en octubre de 2009.
